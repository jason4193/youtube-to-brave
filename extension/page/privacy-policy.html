Privacy Policy — YouTube to Brave Redirector

Last updated: March 7, 2026

Summary

This extension is designed to run locally on your device.

  • It detects YouTube navigation in Chrome and sends only the URL needed to open Brave.
  • The native host script receives that URL via Chrome Native Messaging and opens Brave locally.
  • No analytics, cloud sync, account login, or external API calls are used.

Data processed by the extension

  • YouTube URLs from youtube.com and youtu.be navigations.
  • Platform type (macOS/Windows/Linux) for install guide UX.
  • Extension and native host versions for compatibility checks.

Installer files are downloaded only when you click the install button in the popup UI.

Data processed by the native host

  • URL messages sent from the extension (expected http / https only).
  • Version-check messages.

Storage and retention

Default mode: no persistent logging of browsing data.

  • Extension: no telemetry/history database is maintained.
  • Native host: no log file is written unless debug mode is explicitly enabled.

Debug mode (off by default)

If debug mode is manually enabled (YTB_DEBUG=1 or debug marker file), diagnostic logs are written locally to ~/youtube_redirect_debug.log.

These logs are stored only on your device and can be deleted any time.

What we do not do

  • Do not transmit URLs to external servers.
  • Do not sell or share personal data with third parties.
  • Do not execute remote code.
